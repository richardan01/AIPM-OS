# PM decisions log — template

Format reference for `pm-decisions-log.md` (the live log is local-private and gitignored). Copy this structure into your local `pm-decisions-log.md`; the monthly `Workflows/memory-consolidation.md` audit promotes durable AI PM decisions here in PD-TOL form.

Each entry uses **PD-TOL**: Problem → Decision → Tradeoff → Outcome → Learning.

## Entry format

### YYYY-MM-DD — <short decision title>
- **Problem:** <the decision context — what forced a choice>
- **Decision:** <what was decided>
- **Tradeoff:** <what was given up; the B / C / D options not taken>
- **Outcome:** <result, once known — leave as "pending" until observed>
- **Learning:** <what this teaches for the next call; interview-story signal if any>
