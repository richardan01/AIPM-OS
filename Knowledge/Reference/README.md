# Knowledge/Reference

Canonical reference context that doesn't change week to week — reference architecture, stable domain facts, and durable ground-truth the AI PM mission relies on. Distinct from `Knowledge/Concepts/` (cross-cutting patterns and mental models) and `Knowledge/Research/` (raw research outputs).

**Private by design.** Reference contents are gitignored (`.gitignore`: `Knowledge/Reference/*`) — they may include private or sensitive facts and are never committed to the public repo. This README is the only tracked file in the folder.

## Convention
- One file per reference topic.
- Update when the underlying fact changes; the monthly `Workflows/memory-consolidation.md` audit checks for staleness.
