# Knowledge/People

Stakeholder and network profiles for the AI PM mission — frontier-lab staff, hiring managers, warm-intro paths, and other contacts whose context matters for interview stories and network warming.

**Private by design.** The profiles themselves are gitignored (`.gitignore`: `Knowledge/People/*`) — they hold personal contact detail and are never committed to the public repo. This README is the only tracked file in the folder; a fresh clone gets the folder and its purpose, not the contents.

## Convention
- One file per person: `<name>.md`.
- Edit existing profiles only with confirmation (see `CLAUDE.md` DO-NOT).
- Owner: Oracle files here after recon; Gordon runs warm-intro path-finding.
