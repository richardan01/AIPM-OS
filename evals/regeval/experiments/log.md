## 2026-06-28 — forward discovery pass + binary collapse (see discovery-pass-2026-06-28.md)

**Forward open→axial coding** on the actual disagreement set surfaced two integrity defects:
- **FM-11 phantom held-out:** `gold_expansion.jsonl` (70) = exact subset of `gold.jsonl` (reg-0031..0100). The 2026-06-28 "held-out confirm" κ=0.661 was in-sample. No valid out-of-sample measurement existed.
- **FM-12 unclear fails IAA:** blind 2nd annotator agreed compliant 13/13, non-compliant 13/13, **unclear 2/18** → IAA κ=0.480 (<0.60). `unclear` is unreliable gold, not just hard.

**Decision:** collapse to **binary** judge (drop `unclear`). `unclear` 18 items quarantined (split tag).

**Runs (binary-v1, Sonnet subagent, forced-commit, shuffled order):**
- `02` in-sample N=82 → **κ=1.000** (TPR 42/42, TNR 40/40). Beat the old 3-class run *on the binary items* (old: 3 errors + 11 abstentions) — the broken unclear triggers were degrading binary too.
- `03` GENUINE held-out `heldout_v2.jsonl` N=36 (20 new regimes, dimension-sampled) → **κ=1.000** (TPR 18/18, TNR 18/18). First valid generalization evidence.
